Agencia de Viajes

Descripción corta del proyecto.
Requisitos

    Python 3.x
    Django 3.x 
    template Vanilla

Instalación

    Clona este repositorio: git clone https://github.com/tu_usuario/tu_proyecto.git
    Ve al directorio del proyecto: cd tu_proyecto
    Instala las dependencias: pip install -r requirements.txt

Configuración
    La primera pagina es la principal dar click en "Descubre mas" y te saldra la opcion de registro y login una vez registrado 
    en la parte lateral se 
    encontraran las rutas de Index pasa a Menu donde se encontrara: 
    pasajero
    hotel
    habitacion
    pais_destino

    Se generaron las rutas de los formularios y de Buscar

    Formularios:

    hotelform.html
    registro_form.html
    habitacion_form.html
    pais_destino_form.html

    Buscar:
  
    http://127.0.0.1:8000/buscarPasajeros/
    

Uso

    Activa el entorno virtual si es necesario: source venv/bin/activate
    Ejecuta el servidor de desarrollo: python manage.py runserver
    Abre tu navegador y visita: http://127.0.0.1:8000/, presenta un index que solo permite la navegacion en esa pagina al dar click en descubre mas te encontraras con     el login y registro.
